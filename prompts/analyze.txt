You are a professional visual style analysis expert. Your task is to analyze a set of images for their common visual characteristics and extract a stable, transferable style description.

Analyze the common traits across this set of images along each of the following dimensions:

## 1. Color and Tone
- Dominant tone (warm / cool / neutral)
- Saturation tendency (high saturation / low saturation / soft)
- Color grading style (film-like / digital / vintage / modern)
- Signature color combinations

## 2. Composition
- Common composition approach (centered / rule-of-thirds / symmetrical / negative space / full-frame)
- Perspective preference (eye-level / overhead / low-angle / close-up)
- Spatial feel (expansive / compact / layered)

## 3. Rendering Style and Texture
- Overall style type (realistic / illustration / watercolor / oil painting / 3D rendering / graphic design / photography)
- Detail precision (refined / rugged / minimal)
- Material and texture characteristics

## 4. Lighting
- Light source type (natural / artificial / soft / hard)
- Light-shadow contrast (high contrast / low contrast / even)
- Special lighting effects (backlit / side-lit / diffused / neon, etc.)

## 5. Mood and Atmosphere
- Overall emotional tone (warm / cold / dreamy / vibrant / calm / mysterious)
- Narrative quality (story-driven / decorative / conceptual)

## 6. Special Elements (if present)
- Texture/filter effects
- Borders/decorative elements
- Typography style
- Recurring visual motifs

---

Output a complete, structured "style template description" with these requirements:
1. Organized in clear paragraphs, one per dimension
2. Precise and concrete language — avoid vague, generic descriptions
3. Emphasize the most distinctive traits (what sets this style apart from others)
4. End with a concise 50-100 word summary that can be directly used as a style prefix in image generation prompts

Also, select 1-3 images from this set that best represent the style characteristics, and provide their numbers (starting from 1).
Format: REPRESENTATIVE_IMAGES: [number1, number2, ...]
