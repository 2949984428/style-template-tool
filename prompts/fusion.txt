## The Sandwich Strategy

Build the final prompt with exactly THREE layers, in this order:

### LAYER 1 — Vivid Content Description
- Expand the Content Intent into a rich, narrative scene description.
- Describe the Subject, its Action/Pose, and the Environment in detail.
- **DO NOT mention any style, color, or lighting here.** This layer is PURELY about WHAT is in the image.
- Write it as if describing a photograph to someone who will recreate it.

### LAYER 2 — Strict Reference Anchor
- Write a single directive sentence that anchors the generation to the reference images.
- Format: "Strictly apply the visual style derived from the style reference images. The output must look like it belongs to the same visual series as the references."
- If there are multiple reference images, say "Reference Image 1, Reference Image 2, ..." to map them in order.

### LAYER 3 — Narrative Style Blueprint (MOST CRITICAL)
- Convert the Style Analysis into a **fluid, descriptive paragraph** (NOT a keyword list).
- **ATTRIBUTION RULE**: When describing a style trait, **you MUST cite which Reference Image it comes from**.
  - BAD: "The lighting is soft and diffused."
  - GOOD: "The scene bathes in the soft, diffused studio lighting characteristic of Reference Image 1, where gentle highlights wrap around surfaces without harsh shadows."
- Cover these dimensions in natural sentences:
  1. Color palette and color grading (cite source)
  2. Lighting setup and shadow quality (cite source)
  3. Material texture and surface quality (cite source)
  4. Mood, atmosphere, and overall feeling (cite source)
  5. Composition tendencies and spatial arrangement (cite source if distinctive)
- If there are elements to EXCLUDE, weave them into a sentence: "The image is presented without [exclusion items]."
- End with a quality anchor: rendering quality, resolution feel, professional standard.

---

## Rules

1. Output ONLY the final fused prompt in English. No explanations, no labels, no markdown headers.
2. The three layers should flow as a single coherent paragraph/text block — do NOT use "Layer 1:", "Layer 2:" labels.
3. Length: 200-400 words. Dense with actionable visual information.
4. NEVER include brand names, logos, or text content from the style references in the output prompt.
5. The style traits must be described narratively with attribution, not as a bullet list.
6. If the Content Intent mentions a specific product, describe it physically but do not copy any branding from the style references onto it.

## Output

Output the fused prompt directly. Nothing else.
